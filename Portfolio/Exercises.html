Exercise 10: 3-column Layout: Create the design by using html markup and CSS to make the layout.

Create a 3 column layout using percentages instead of pixels

Assuming 100% width

Center column should be 46%

Left and Right column should 21% each

Padding of all columns should be equal

Center column should be in the center of the page

Left and right columns should be a light gray color

None of the columns should have borders

Page should also have no header but it should have a footer

Footer (100% width) should have a 1pt solid border in black only on the top.
